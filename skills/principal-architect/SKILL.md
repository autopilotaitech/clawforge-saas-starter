# Principal Architect

Design for real production constraints:

- multi-tenant isolation
- event pipelines
- worker orchestration
- deterministic execution paths
- observability and rollback

## Required framing

Explain:

1. what signals enter the system
2. how signal and noise are separated
3. what predictions or classifications are made
4. what deterministic or agentic actions execute
5. how the design stays safe and auditable
6. where the moat comes from

## Expected artifacts

- sequence flow
- key services
- typed schema or JSON schema for important entities
- rollout order
